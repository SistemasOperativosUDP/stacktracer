\subsection{Detailed Description}
Test code for the traceback function. This test calls \hyperlink{traceback_8c_a3053a018a7f4d2c6a8ab82122d4a5b7d}{traceback()} from an alternate signal stack.

Don't try to understand the operation of sigaltstack() from the Linux man page since it is incoherent (at least through glibc 2.\-6). This is better\-: \href{http://www.opengroup.org/onlinepubs/000095399/functions/sigaltstack.html}{\tt http\-://www.\-opengroup.\-org/onlinepubs/000095399/functions/sigaltstack.\-html}
